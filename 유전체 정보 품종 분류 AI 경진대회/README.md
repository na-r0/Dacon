## 주제: 유전체 정보 품종 분류 AI 경진대회

기간: 2022.12.12 ~ 2023.01.16

#### Dataset Info.
    - id : 개체 고유 ID  
    - 개체정보  
        - father : 개체의 가계 고유 번호 (0 : Unknown)  
        - mother : 개체의 모계 고유 번호 (0 : Unknown)  
        - gender : 개체 성별 (0 : Unknown, 1 : female, 2 : male)  
        - trait : 개체 표현형 정보   
    - 15개의 SNP 정보 : SNP_01 ~ SNP_15  
    - class : 개체의 품종 (A,B,C)
